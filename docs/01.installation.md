# Installation

Install the bundle in your project directory:

```shell
$ npm install --save @conga/framework-rest
```

Enable the bundle in your application configuration:

```yaml
    // app/config/bundles
    bundles:

        all:
            - "@conga/framework-rest"
```

```js
    /**
     * This is a comment
     *
     * @Route("/foo/bar")
     */
    foo(req, res) {
        console.log('hello')
    }
```
